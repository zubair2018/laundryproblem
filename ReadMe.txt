# Laundry Wallah / Laundry Mart

## Project Description
This is a simple HTML website for a laundry service business. It includes information about services, pricing, and a booking form.

## Features
- About Us section
- Image gallery
- Services list
- Price table
- Contact form

## How to Run
1. Open `index.html` in any web browser.
2. The website will load and display all sections.

## Deployment
This project is deployed on Netlify and can be accessed online.
